# QuitFlame — 禁煙サポートアプリ

「吸いたい」衝動が来たときに、厳しい言葉で我に返すための禁煙サポートツール。

## 機能

- **衝動モード**: 「吸いたい！」→ 状況を選ぶ → 厳しめの言葉で喝を入れる
- **禁煙カウンター**: 日数・節約金額・回避本数・延びた寿命をリアルタイム表示
- **健康回復タイムライン**: 禁煙後の体の回復過程を医学的事実に基づいて表示
- **3つのトーン**: 鬼教官 / 毒舌の親友 / 冷静な医者から選択
- **段階的メッセージ**: 禁煙日数と衝動の状況に応じて内容が変わる
- **衝動ログ**: いつ・どんな状況で吸いたくなったかの記録
- **段階的リセット**: 3回確認を経てカウンターリセット（温かく再スタート）

## プライバシー

- **すべてのデータはブラウザ内（IndexedDB）に保存されます**
- 外部サーバーへのデータ送信は一切ありません
- アクセス解析・トラッキングは行っていません
- ブラウザのデータを消去するとすべてリセットされます

## 起動方法

### ローカル（Docker）

```bash
docker compose up --build
# http://localhost:3001
```

## 技術構成

- React + TypeScript + Vite
- Tailwind CSS v4
- IndexedDB (Dexie.js)
- Docker (nginx) / GitHub Pages

## 注意事項

- このアプリは医療アドバイスを提供するものではありません
- 禁煙治療が必要な場合は医療機関への相談を推奨します
- 健康回復タイムラインは一般的な医学的知見に基づいた目安です
- 個人差があります
